
# Aspose complaint form


 ### Installation
Requires [Node.js](https://nodejs.org/) to run.

Install the dependencies and devDependencies and start the server.

```sh
$ cd Aspose
$ npm install 
$ npm  start
go to the browser and hit the URL "http://localhost:8080"
```


 # Flexibility
 - Easy to change.
